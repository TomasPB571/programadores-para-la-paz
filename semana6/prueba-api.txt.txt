200 OK — El servidor recibió la petición sin errores. Es el código de estado HTTP que significa "todo salió bien".

307 B — El tamaño de la respuesta fue de 307 bytes, algo muy liviano.

  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
